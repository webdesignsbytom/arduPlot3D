# TPGL

## Plotting language

Tom's Plotter Graphic Language (TPGL)

## Commands

### Motions

1. Move to = M
2. Finger Down = FD
3. Finger Up = FU
4. End = E
5. XY speed = 
6. Z speed =
7. Number of fingers FN
8. Pause = P

### Data Types

Plot points have 5 data tpyes

1. Tap = DT
2. Move and Tap = DMT
3. Drag = DD
4. Move = DM
5. Time out DTO

### Functions

Test functions and UI commands
Sequence alphabetical after T

1. Test XY = TA
2. Text Z = TB
3. Text XYZ = TC
4. Test finger 1 = TD 
5. Test finger 2 = TE 
6. Test finger 3 = TF
